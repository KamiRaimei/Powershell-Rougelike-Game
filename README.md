## Terminal Rougelike Game
A just for fun repo, originally to test Powershell capabilities in a form of a rougelike game.
Now available in python port for cross platform compatibility.

### Requirements
#### Powershell
Powershell 7 is required

#### Python
No specific requirements.

### How to run
#### Powershell
Run the following command in windows terminal within the script folder

```.\Terminal-Rougelike-Game.ps1```

#### Python
Run the following command in terminal within the script folder (assuming python is installed)

```python ./Terminal-Rougelike-Game.py```
